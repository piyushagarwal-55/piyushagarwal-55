# Resume And Profile Checklist

Use this before sending the profile or resume to recruiters.

## Resume

- Add exact CGPA in the education section.
- Add the GitHub profile link: `https://github.com/piyushagarwal-55`.
- Add 3-4 best project links with live demos, not only source code.
- Mention measurable proof where available: users, latency, model used, number of features, contracts deployed, languages supported, or hackathon result.
- Keep project bullets understandable to non-technical recruiters.

Example education line:

```text
B.Tech, Computer Science / related branch | CGPA: <exact-value>/10
```

## GitHub Profile

- Add LinkedIn URL in GitHub profile sidebar.
- Add portfolio URL in GitHub profile sidebar.
- Pin the strongest 6 repositories:
  - FlowForge
  - GyaanSaarthi
  - ShopSage
  - Nexus Flow, if the source repo is public
  - Eden Haus
  - RepVote
- Add repo descriptions for every pinned project.
- Add repository homepage links for every deployed project.
- Add YouTube demo links to pinned repo READMEs when the app is complex.

## Project Repositories

- Put live demo/video/screenshots before installation steps.
- Add demo credentials where safe.
- Add `assets/screenshots/` folder and commit real screenshots.
- Keep README language simple enough for a recruiter.
- Keep technical depth lower on the first screen and deeper after the demo section.
